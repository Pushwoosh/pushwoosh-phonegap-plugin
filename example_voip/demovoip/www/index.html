Welcome to Pushwoosh VoIP Demo

How to use this app:

  1. Enter your App ID (VoIP App Code) and FCM Project ID
  2. Click "Initialize Pushwoosh" to start the SDK
  3. Initialize VoIP Parameters to configure video support and ringtone
  4. Request Call Permission on Android if needed
  5. Use End Call button to terminate active calls
  6. Monitor VoIP events in the Event Log below

Testing VoIP Calls:

To receive incoming VoIP calls, send a VoIP push notification from the Pushwoosh Control Panel using your App ID.
