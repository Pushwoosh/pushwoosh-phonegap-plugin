Cross-Platform push notifications by Pushwoosh for Cordova / PhoneGap

#### Cordova

Using npm (requires cordova 7.0+):

```
cordova plugin add pushwoosh-cordova-plugin@7.17.1
```

Using git:

```
cordova plugin add https://github.com/Pushwoosh/pushwoosh-phonegap-plugin.git#7.17.1
```

#### Phonegap

Using npm (requires phonegap 7.1+):

```
cordova plugin add pushwoosh-cordova-plugin@7.17.1
```
